# FrontEnd_Pro_Final
Front-End Pro course final test 

Описание и инструкции

Описание:
В репозитории находятся 2 папки: client и nodeServer
В папке client точка входа - это файл index.html

В папке nodeServer:
Точка входа - это файл server.js (с кодом сервера). Сервер по умолчанию слушает порт 4321 (можно изменить в последней строке кода файла server.js);
Так же в папке nodeServer есть файл tasksDatabase.json - это JSON база данных, с которой работает server и куда записывает/изменяет/удаляет/вычитывает созданные таски.
Клиентский script настроен таким образом, что он запрашивает у сервера существующие таски в базе данных и отображает их на клиенте в том числе и в момент  загрузки/обновления страницы. Поэтому в tasksDatabase.json уже есть 2 тестовые записи (просто для демонстрации).
Так же в папке tasksDatabase.json удалена папка node_modules, поэтому после того, как проект будет развернут из git-a в локальном репозитории, node модули нужно установить вручную (на основе файла package.json).

Инструкции
1) Клонировать проект в локальный репозиторий
2) Из командной строки зайти в папку nodeServer
3) Установить node модули из package.json:

	> npm install
	
4) Запустить сервер:

	> node server.js

5) Открыть клиента - index.html в папке client
